箭头函数中this的使用跟普通函数也不一样，在JavaScript的普通函数中，都会有一个自己的this值，主要分为：

  1. 函数作为全局函数被调用时，this指向全局对象
  2. 函数作为对象中的方法被调用时，this指向该对象
  3. 函数作为构造函数的时候，this指向构造函数new出来的新对象
  4. 还可以通过call，apply，bind改变this的指向
